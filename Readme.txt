A personal front-end starter heavily inspired by the awesome Fireshell framework (http://getfireshell.com/).

Requirements:
- Git installed (http://git-scm.com/)
- Ruby installed (http://rubyinstaller.org/)
- Node.js installed (https://nodejs.org/)
- SASS installed (sass-lang.com)
	- Open command prompt
	- Type 'gem install sass' -> Requires Ruby to already be installed
- Bower installed (http://bower.io/)
	- Open command prompt
	- Type 'npm install -g bower' in command prompt -> Requires node.js to already be installed	
- Internet connection

Default modules:
- gulp
- gulp-ruby-sass //To compile SASS into CSS
- gulp-notify
- gulp-bower

Default packages:
- jquery
- bootstrap-sass
- fontawesome
